## 🔧 Quick Reference

**🚀 Start the platform:**
```bash
docker compose up -d
```

**🛑 Stop the platform:**
```bash
docker compose down
```

**📊 Check service health:**
```bash
docker compose ps
```

**📝 View logs:**
```bash
docker compose logs [service-name]
```

**🔄 Restart a service:**
```bash
docker compose restart [service-name]
```
